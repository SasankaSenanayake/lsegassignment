To run this scripts;

1. Only you have to put this script1.yml file and script2.yml file into ansible installed environment.
2. Configure ssh and copy key to the webserver. (13.233.215.86)
3. Add the webserver ip to ssh hosts file.
4. Run script 1 using ansible-playbook script1.yml
5. Run script 2 using ansible-playbook script2.yml
